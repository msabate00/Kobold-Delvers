# Kobold Delvers

Small 2D puzzle game made in **C++17** with **OpenGL**.

The game is built on top of **SandForge**, my own small falling-sand engine. The engine code is still inside the repository, but this project is now focused on the actual game.

## What is Kobold Delvers?

Kobold Delvers is a puzzle game where you guide kobolds through dangerous caves using simulated materials.

Most of the time you do not control the kobolds directly. They walk, fall, drown, burn and get stuck by themselves. Your job is to paint materials into the level and use the simulation to help them reach the goal.

Some levels are different and let you control one kobold directly.

## Gameplay

The main idea is simple:

- kobolds spawn from the entrance
- they try to move through the level on their own
- you can place materials with a limited budget
- the level is completed when enough kobolds reach the goal
- some levels have a bonus to collect
- using fewer materials can give an extra star

Every fourth level is a special level where you control the kobold directly instead of only helping from outside.

## Game modes

### Levels

The game has **12 handmade levels**.

- 9 normal levels
- 3 special player-controlled levels
- stars and progress saving
- level thumbnails in the selector
- unlocks based on the stars you have

### Sandbox

There is also a sandbox mode to test materials and create custom levels.

In sandbox you can:

- paint any material
- place kobolds, spawners, goals, bonus items and player triggers
- change the level size
- change the material budget
- save and load custom `.lvl` files
- undo and redo changes with `Ctrl + Z` / `Ctrl + Y`

Custom levels are saved in:

```text
levels/custom/
```

## Materials

The game uses a cellular falling-sand simulation. Some materials are simple, and some react with others.

Current materials/entities include:

| Material | Use / behavior |
| --- | --- |
| Empty | Removes cells and lets you clean parts of the level. |
| Sand | Falls, piles up and can bury entities. |
| Water | Flows, cools hot coal and reacts with lava. |
| Stone | Basic solid material for building or blocking paths. |
| Wood | Flammable solid material. |
| Fire | Burns flammable materials and triggers reactions. |
| Lava | Hot, dangerous and reactive material. |
| Smoke | Gas generated by fire. |
| Steam | Rising gas generated by heat interactions. |
| Vine | Grows with water, burns with fire and can be used as support. |
| Snow | Falls and melts with heat. |
| Oil | Flammable liquid. |
| Coal / Hot coal | Coal can ignite and spread heat or fire around it. |
| Flammable gas | Gas that can explode when ignited. |
| Fragile platform | Platform that can break when entities step on it. |
| Acid | Corrosive liquid that consumes some materials. |
| Dynamite | Explodes when it touches heat or fire. |

## Progress

Progress is saved in:

```text
progress.cfg
```

Normal levels can give up to 3 stars:

1. complete the level
2. collect the bonus
3. complete it under the budget star limit

Special levels give 1 star when completed.

  game/          scenes, menus, level selector, sandbox
